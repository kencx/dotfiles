# dotfiles

## Install
Run `install.sh` to set up a new env 

## Folder Structure
The folder structure is set up for [stow](https://www.gnu.org/software/stow/) as the dotfiles manager. Refer to
[this](https://www.gnu.org/software/stow/) guide for how it works.
